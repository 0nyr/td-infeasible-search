# td-infeasible-search

Standalone experiments for a research study: **penalised infeasible search-space exploration for time-dependent vehicle routing (TDVRPTW/TDVRP) under exact arithmetic** — a time-warp analogue defined on NDCPWLF (non-decreasing continuous piecewise-linear) arrival-time-function compositions, with no epsilons and bitwise reduction to the reference checker on feasible routes.

Part of Onyr's PhD KAYROS programme. Sibling of [td-route-trees](https://github.com/0nyr/td-route-trees) (the TD local-search structures study that produced the KAYROS TD-LS layer). The production implementation of the TD time-warp is developed in [kayros](https://github.com/0nyr/kayros) (branch `td-time-warp` until integration); the study's design memos, session logs, and experiment reports live in the private research workspace. This repo hosts targeted standalone implementation experiments if and when the study needs them.

## Status

No standalone experiments yet (2026-07-07). The study's design phase is complete; prototyping happens in kayros.
